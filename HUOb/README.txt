Features:
=========
a) Visitors Online script that counts the total users online in a web page.
b) This free Visitor counter script works based on a time interval of 300 seconds.
c) This simple web site visitors online counter also displays the count of maximum visitors a web page had at a time.
c) Simple file based utility.
d) Easy to use.


Installation:
=============
Please take 5 minutes time and read installation instructions carefully and
completely! This will ensure a proper and easy installation.

a) Unzip HUOb_1_0.zip. to extract the files to the  folder HUOb_1_0\HUOb

File Permissions:
 a) Set proper read write permissions (666) for ip.txt (Only for Linux Users).

Embedding the Visitors Online Counter:
 a) Open the file HUOb/about.php.
 b) Click on the option “Get Code”
 c) Click on “Select All” to copy the code.
 d) Change your file name as .php (e.g: index.html to index.php).
